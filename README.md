# SBRP Application

This is a full-stack web application built to support the creation, management, and reporting of **Small Business Restructuring Proposals**.

It includes a Node.js and Express backend, a React + Redux frontend, and a MongoDB database. The application streamlines client onboarding, data collection, case tracking, and report generation ‚Äî making it easier for professionals to manage restructuring processes.
